================================================================================
                    HTML & CSS CHEATSHEET (LEARNED + ADVANCED)

--- HTML: PAGE STRUCTURE & METADATA ---
<!DOCTYPE html>        - Tells the browser this is an HTML5 document. 
<html>                 - The root element that wraps all content on the page. 
<head>                 - A container for metadata (data about the page) that isn't visible. 
<meta charset="UTF-8"> - Ensures special characters and symbols display correctly. 
<meta name="viewport"> - Sets the visible area of the page; essential for mobile responsiveness. 
<title>                - The text shown on the browser tab or in search results. 
<link>                 - Connects your HTML to external files, like your CSS stylesheet, or favicon, or fonts and icons. 
<body>                 - Contains all the visible content (text, images, buttons). 

--- HTML: CONTENT HIERARCHY & TEXT ---
<h1> ... <h6>          - Section headings (h1 is the most important, h6 is the least). 
<p>                    - Defines a block of text as a paragraph. 
<strong>               - Makes text bold and indicates it has "strong importance." 
<b>                    - Makes text bold for visual style without extra importance. 
<em>                   - Italics that indicate emphasis when reading. 
<i>                    - Italics for visual style (like technical terms or thoughts). 
<small>                - For "fine print" like legal disclaimers or copyright text.
<hr>                   - A horizontal rule (line) to separate content sections.
<address>              - Contact information for author/owner. Usually italicized.
  *Attributes: Global attributes only

--- HTML: LAYOUT CONTAINERS (SEMANTIC) ---
<main>                 - Wraps the unique, central content of your webpage. 
<header>               - The introductory section of a page or a specific area. 
<section>              - Groups related content together (e.g., an "About Us" section). 
<footer>               - The bottom section of a page, usually for contact or links. 
<div>                  - A generic "box" used to group elements for styling (Block-level). 
<span>                 - A generic container for small bits of text inside a line (Inline). 

--- HTML: LISTS, LINKS & MEDIA ---
<ul>                   - Creates a bulleted (unordered) list. 
<ol>                   - Creates a numbered (ordered) list. 
<li>                   - An individual item inside a <ul> or <ol>. 
<a>                    - A hyperlink. Use the 'href' attribute to set the destination URL. 
  *Attributes: 
    download            - Prompts download instead of navigation
    target              - Where to open link: "_self" (default), "_blank" (new tab/window), 
                         "_parent" (parent frame), "_top" (full window), or frame name
<img>                  - Displays an image. Needs 'src' (path) and 'alt' (description).
<nav>                  - A section intended for navigation links (menus).
<article>              - For self-contained content (like a blog post or news story).
<figure> / <figcaption>- Groups an image/diagram with its descriptive caption.
<video> / <audio>      - Used to embed playable movie or sound files. 
<iframe>               - Displays an entire different webpage inside a window on yours.
<source>               - Media resource for <picture>, <audio>, <video>.
  *Attributes: src, srcset, sizes, media, type
<embed>                - Embeds external content/plugin (Flash, PDF viewer).
  *Attributes: src, type, width, height

--- HTML: FORMS & INTERACTION ---
<form>                 - A container that collects user input to send to a server. 
<label>                - Provides a clickable text description for an input field. 
<button>               - A clickable button to submit forms or trigger actions. 
<select>               - Creates a drop-down selection menu. 
<option>               - An individual choice inside a <select> drop-down.
<textarea>             - A larger, multi-line box for long user comments.
<fieldset> / <legend>  - Groups related inputs together with a visible title.
<input> types:
  type="email"         - Email address with validation
  *Additional types: text, password, checkbox, radio, submit, reset, button
<input> attributes:
  *placeholder         - Hint text in empty field
  *required            - Field must be filled before submission (boolean attribute)
  *Additional attributes: name

--- HTML: DEFINITION LISTS ---
<dl>                   - A list of terms and their specific descriptions. 
<dt>                   - The specific "Term" or name being defined. 
<dd>                   - The detailed description or definition for the term.

--- HTML: TABLES ---
<table>                - The main container for tabular data. 
<thead> / <tbody>      - Groups the "Header" rows vs the "Main Content" rows. 
<tr>                   - Defines a single horizontal "Table Row." 
<th>                   - A "Table Header" cell (usually bold/centered). 
<td>                   - A standard "Table Data" cell. 
<caption>              - A title or description for the entire table.

--- HTML: GLOBAL ATTRIBUTES (WORK ON ALL ELEMENTS) ---
*class                 - Space-separated CSS class names for styling
*id                    - Unique identifier for element (must be unique per page)
*style                 - Inline CSS styles (direct styling on element)

--- CSS: TEXT & TYPOGRAPHY ---
color                  - Sets the color of the text. 
font-family            - Defines which font/typeface to use (e.g., Arial, sans-serif). 
font-size              - Sets how large the text appears. 
font-weight            - Controls the thickness of text (e.g., bold, 400, 700). 
font-style             - Used to make text italic or normal. 
text-align             - Moves text to the left, center, right, or justifies it. 
text-decoration        - Adds or removes lines (e.g., underline, line-through, none). 
line-height            - Adjusts the vertical spacing between lines of text. 
letter-spacing         - Adjusts the horizontal space between individual characters. 
::first-line           - Styles first line of text in an element
::first-letter         - Styles first letter of text in an element

--- CSS: THE BOX MODEL (SPACING & SIZING) ---
width / height         - Sets the specific dimensions of an element's content box. 
min-width / max-width  - Ensures a box never gets smaller/larger than a certain point. 
padding                - Space *inside* the box, between the content and the border. 
border                 - The line around the padding (Set: width, style, color). 
border-radius          - Rounds the corners of an element's outer border. 
margin                 - Space *outside* the box, creating distance from other elements. 
display                - Defines how the box behaves (block, inline, flex, grid). 
box-sizing             - Controls how width/height are calculated
  *Values:             - content-box (default), border-box (includes padding/border)
display values:
  *block              - Takes full width, forces line break before/after
  *inline             - Flows with text, ignores width/height
  *inline-block       - Flows inline but respects width/height/margin/padding
  *flex               - Flexbox container
  *grid               - Grid container
  *none               - Completely removed from flow
  

--- CSS: FLEXBOX & GRID (MODERN LAYOUT) ---
display: flex          - Aligns items in a row or column with powerful spacing tools. 
justify-content        - Aligns items along the main axis (horizontal by default). 
align-items            - Aligns items along the cross axis (vertical by default). 
gap                    - Sets a specific distance between items in a Flex/Grid layout.
display: grid          - Creates complex layouts using both rows and columns.

Flexbox properties:
flex-direction         - Main axis direction: row (default), row-reverse, column, column-reverse
flex-wrap              - Wrapping: nowrap (default), wrap, wrap-reverse
flex-flow              - Shorthand for flex-direction and flex-wrap
justify-content        - Main axis alignment: flex-start, flex-end, center, space-between, space-around, space-evenly
align-items            - Cross axis alignment: stretch, flex-start, flex-end, center, baseline
align-content          - Multi-line alignment: flex-start, flex-end, center, space-between, space-around, stretch
align-self             - Individual item alignment: auto, flex-start, flex-end, center, baseline, stretch

Grid properties:
grid-template-columns  - Defines column track sizes: none, auto, length, %, fr, min-content, max-content, minmax(), repeat()
  *Example: grid-template-columns: 100px 1fr 2fr;
grid-template-rows     - Defines row track sizes (same values as columns)
grid-template-areas    - Named grid areas using strings
  *Example: grid-template-areas: "header header" "sidebar main" "footer footer";
grid-column-start      - Start line for column: line number, span n, auto
grid-column-end        - End line for column
grid-row-start         - Start line for row
grid-row-end           - End line for row
grid-column            - Shorthand: start / end
grid-row               - Shorthand: start / end
grid-area              - Shorthand: row-start / column-start / row-end / column-end OR area name
justify-items          - Horizontal alignment in grid cell: stretch, start, end, center
align-items           - Vertical alignment in grid cell: stretch, start, end, center
justify-content       - Horizontal alignment of grid tracks: start, end, center, stretch, space-around, space-between, space-evenly
align-content         - Vertical alignment of grid tracks (same values as justify-content)

Grid functions:
*repeat()            - Repeats track pattern: repeat(3, 1fr) = 1fr 1fr 1fr

--- CSS: POSITIONING & LAYOUT ---
position               - How an element sits in the flow (relative, absolute, fixed, sticky). 
  *Values:
    static            - Default. Element flows normally (cannot use top/right/etc.)
    relative          - Positioned relative to its normal position
    absolute          - Removed from flow, positioned relative to nearest positioned ancestor
    fixed             - Removed from flow, positioned relative to viewport (stays on scroll)
    sticky            - Hybrid: acts relative until scroll threshold, then fixed
top/bottom/left/right  - Moves a "positioned" element from its original spot.
z-index                - Controls which element sits "on top" if they overlap (higher = top).

--- CSS: BACKGROUNDS ---
background-color       - Fills the background of an element with a solid color. 
background-image       - Sets an image to sit behind the content. 
background-size        - Controls if the image covers the box or fits inside it. 
background-repeat      - Determines if the image tiles/repeats if it's smaller than the box. 
background-position    - Moves the background image (e.g., center, top-right). 
background             - Shorthand for all background properties
*url()                 - References external resource for background-image
  *Example: background-image: url("image.jpg");
*linear-gradient()     - Linear color gradient for backgrounds
  *Example: background: linear-gradient(to right, red, blue);

--- CSS: SPECIAL EFFECTS ---
box-shadow             - Adds a drop-shadow effect behind an element's frame.
opacity                - Sets how transparent an element is (0.0 to 1.0).
z-index                - Controls which element sits "on top" if they overlap.
column-gap             - Gap between columns in multi-column layout or grid
  *Also: row-gap, gap (shorthand for row-gap and column-gap)

--- CSS: INTERACTIVITY ---
:hover                  - Styles an element only when the mouse is over it.
:cursor                 - Changes the mouse icon (e.g., to a "hand" pointer)
:focus                  - Styles an element (like an input) when it's clicked into.

--- CSS: SELECTORS (EXPANDED) ---
Attribute selectors:
  [attribute]          - Elements with attribute
    *Example: [disabled] selects all disabled elements
  [attribute="value"]  - Exact attribute value
    *Example: [type="email"] selects all email inputs
  [attribute~="value"] - Attribute contains word (space-separated)
    *Example: [class~="btn"] selects elements with class containing "btn"
  [attribute|="value"] - Attribute starts with word or word-
    *Example: [lang|="en"] selects lang="en" or lang="en-US"
  [attribute^="value"] - Attribute starts with string
    *Example: [href^="https"] selects links starting with "https"
  [attribute$="value"] - Attribute ends with string
    *Example: [src$=".jpg"] selects images ending with ".jpg"
  [attribute*="value"] - Attribute contains substring
    *Example: [href*="example"] selects links containing "example"

Pseudo-classes:
  :first-child         - First child element
  :last-child          - Last child element
  :nth-child(n)        - nth child element (even, odd, 2n, 3n+1)
    *Examples: 
      :nth-child(2)    - Second child
      :nth-child(odd)  - All odd children
      :nth-child(3n)   - Every third child
      :nth-child(3n+1) - First child of each group of three
  :nth-last-child(n)   - nth child from end
  :root                - Root element of document (<html>)

Pseudo-elements:
  ::before             - Inserts content before element (requires 'content' property)
    *Example: .element::before { content: "→ "; }
  ::after              - Inserts content after element (requires 'content' property)
  ::first-line         - Styles first line of text
  ::first-letter       - Styles first letter of text

--- CSS: UNITS & VALUES ---
Absolute lengths:
  *px                  - Pixels (1/96th of 1in) - Most common for screens

Relative lengths:
  *em                  - Relative to element's font-size (1em = current font-size)
    *Example: If font-size is 16px, 2em = 32px
  *rem                 - Relative to root element's font-size (1rem = root font-size)
    *Example: If html font-size is 16px, 1.5rem = 24px
  *%                   - Percentage of parent element's corresponding dimension
  *vw                  - 1% of viewport width (100vw = full viewport width)
  *vh                  - 1% of viewport height (100vh = full viewport height)
  *vmin                - 1% of viewport's smaller dimension
  *vmax                - 1% of viewport's larger dimension
  *ch                  - Width of "0" character in current font

--- CSS: COLOR FUNCTIONS ---
*rgb(r, g, b)        - Red, Green, Blue (0-255)
  *Example: rgb(255, 0, 0) = red
*rgba(r, g, b, a)    - RGB with alpha (opacity 0-1)
  *Example: rgba(255, 0, 0, 0.5) = semi-transparent red
*hex                 - #RRGGBB or #RGB (shorthand)
  *Examples: #ff0000 = red, #f00 = red (shorthand), #ff000080 = semi-transparent red

Named colors:
  *red, green, blue, black, white, gray, silver, yellow, purple, orange, pink, 
   brown, cyan, magenta, teal, navy, maroon, olive, lime, aqua, fuchsia

--- CSS: FUNCTIONS ---
*calc()              - Mathematical expressions
  *Examples: 
    calc(100% - 20px)
    calc(50vw + 10rem)
    calc(100vh - 80px)
*url()               - References external resource
  *Example: background-image: url("image.jpg")
*linear-gradient()   - Linear color gradient
  *Examples:
    linear-gradient(red, blue)
    linear-gradient(to right, red, blue)
    linear-gradient(45deg, red, blue)
    linear-gradient(to bottom right, red, blue)
*radial-gradient()   - Circular gradient
*conic-gradient()    - Color wheel gradient
*repeating-linear-gradient() - Repeating linear gradient

  *Example: font-size: clamp(1rem, 2.5vw, 2rem) = font between 1rem and 2rem
*var(--name, fallback)- CSS custom property variable
  *Example: color: var(--primary-color, blue)
*attr()              - Gets attribute value from element
  *Example: content: attr(data-tooltip)

--- CSS: ADVANCED POSITIONING & LAYOUT ---

float / clear          - Older method for wrapping text around images.
  *float values:       - left (floats left), right (floats right), none (default), inherit
  *clear values:       - left (clear left floats), right (clear right floats), both (clear both), none (default), inherit

overflow               - Handles what happens if content is too big for its box.
  *overflow values:    
    visible           - Content overflows outside box (default)
    hidden            - Content clipped, no scrollbars
    scroll            - Content clipped, scrollbars always visible
    auto              - Browser decides (usually scrollbars when needed)
    clip              - Similar to hidden but no programmatic scrolling
  *overflow-x         - Horizontal overflow only
  *overflow-y         - Vertical overflow only

visibility             - Controls element visibility without affecting layout.
  *Values:            
    visible           - Element is visible (default)
    hidden            - Element is invisible but takes up space
    collapse          - For table rows/columns; removes space (like display: none)

clip-path              - Clips element to custom shape.
  *Values:
    circle()          - circle(radius at x y)
    ellipse()         - ellipse(rx ry at x y)
    polygon()         - polygon(x1 y1, x2 y2, ...)
    url()             - References SVG clipPath
    inset()           - inset(top right bottom left round border-radius)

shape-outside          - Defines shape for text wrapping around float.
  *Values: same as clip-path plus: margin-box, border-box, padding-box, content-box

--- CSS: VISUAL EFFECTS & ANIMATION ---

filter                 - Applies effects like blur, grayscale, or brightness.
  *filter functions:
    blur(px)           - Gaussian blur: blur(5px)
    brightness(%)      - Brightness: brightness(150%) = 1.5x brighter
    contrast(%)        - Contrast: contrast(200%) = 2x more contrast
    drop-shadow()      - Drop shadow: drop-shadow(2px 2px 5px black)
    grayscale(%)       - Grayscale: grayscale(100%) = completely gray
    hue-rotate(deg)    - Hue rotation: hue-rotate(90deg)
    invert(%)          - Color inversion: invert(100%) = negative
    opacity(%)         - Transparency: opacity(50%) = 50% transparent
    saturate(%)        - Saturation: saturate(200%) = 2x more saturated
    sepia(%)           - Sepia tone: sepia(100%) = full sepia
    url()              - SVG filter

  *Multiple filters: filter: blur(2px) grayscale(50%) brightness(120%);

transform              - Moves, rotates, scales, or skews an element.
  *2D transform functions:
    translate(x, y)    - Moves element: translate(50px, 100px)
    translateX(x)      - Horizontal movement: translateX(50px)
    translateY(y)      - Vertical movement: translateY(100px)
    scale(x, y)        - Resizes: scale(1.5) or scale(2, 1.5)
    scaleX(x)          - Horizontal scaling: scaleX(1.5)
    scaleY(y)          - Vertical scaling: scaleY(1.5)
    rotate(deg)        - Rotation: rotate(45deg)
    skew(x, y)         - Skew distortion: skew(20deg, 10deg)
    skewX(x)           - Horizontal skew: skewX(20deg)
    skewY(y)           - Vertical skew: skewY(10deg)
    matrix(a,b,c,d,e,f)- 6-value transformation matrix

  *3D transform functions:
    translate3d(x,y,z)
    scale3d(x,y,z)
    rotate3d(x,y,z,angle)
    perspective(length)

  *Multiple transforms: transform: rotate(45deg) scale(1.5) translateX(50px);

transform-origin       - Origin point for transformations.
  *Values:             
    Keyword: left, center, right, top, bottom
    Length: 50px, 20%
    Percentage: 50% 50% (default center)
    Example: transform-origin: top left;

transition             - Animates a CSS change so it happens smoothly over time.
  *Properties:
    transition-property - Which properties to animate: 
                         all, none, or specific properties (width, opacity, etc.)
    transition-duration - How long animation lasts: 0.5s, 200ms, 1s
    transition-timing-function - Animation curve:
                         ease (default), linear, ease-in, ease-out, ease-in-out,
                         cubic-bezier(n,n,n,n), steps(n), step-start, step-end
    transition-delay   - Delay before animation starts: 0s, 1s, 200ms

  *Shorthand: transition: property duration timing-function delay;
  *Example: transition: width 0.3s ease-in-out 0.1s, opacity 0.5s linear;

animation              - Creates complex, multi-step moving effects.
  *Properties:
    animation-name     - Name of @keyframes rule
    animation-duration - Total animation time: 1s, 500ms
    animation-timing-function - Same as transition timing functions
    animation-delay    - Start delay: 0s, 1s
    animation-iteration-count - Number of repeats: 1, 3, infinite
    animation-direction - normal, reverse, alternate, alternate-reverse
    animation-fill-mode - Styles before/after: none, forwards, backwards, both
    animation-play-state - running, paused

  *Shorthand: animation: name duration timing-function delay iteration-count direction fill-mode play-state;
  *Example: animation: slide-in 1s ease-in-out 0s 1 normal forwards;

@keyframes             - Defines animation sequence.
  *Syntax:
    @keyframes animation-name {
      0% { /* starting styles */ }
      50% { /* midpoint styles */ }
      100% { /* ending styles */ }
    }
    
    @keyframes slide-in {
      from { transform: translateX(-100%); }
      to { transform: translateX(0); }
    }

--- CSS: ADVANCED SELECTORS ---

:valid / :invalid      - Styles form elements based on validation.
:required / :optional  - Styles required/optional form elements.
:in-range / :out-of-range - Styles number/range inputs based on min/max.
:read-only / :read-write - Styles based on editability.
:default               - Styles default option in select or radio group.
